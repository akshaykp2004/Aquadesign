🔧 ADVANCED CUSTOMIZATION
═══════════════════════════════════════════════════════════════════════════════

CHANGING COLORS:
  Open style.css and modify CSS variables at the top:
  
  :root {
    --primary-dark: #0A2540;      /* Main dark blue */
    --aqua-teal: #2EC4B6;         /* Accent teal */
    --soft-sand: #F4F1DE;         /* Light sand */
    --coral-accent: #FF7F50;      /* Coral accent */
    /* ... other colors ... */
  }

CHANGING FONTS:
  Update the font imports in each HTML file:
  <link href="https://fonts.googleapis.com/css2?family=YOUR+FONT" rel="stylesheet">
  
  Then update CSS variables:
  --font-display: 'YOUR FONT DISPLAY', serif;
  --font-body: 'YOUR FONT BODY', sans-serif;

ADDING PAGES:
  1. Create new HTML file (e.g., blog.html)
  2. Copy header/footer from any page
  3. Add to navigation menu in navbar
  4. Link it in all footer sections
